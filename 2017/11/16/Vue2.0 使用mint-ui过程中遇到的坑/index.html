Vue2.0 使用mint-ui过程中遇到的坑

发表于 2017-11-16 | 分类于 Vue
字数统计 194 字 | 阅读时长 1 分钟
在学习vue2.0的过程中，搭配mint-ui组件库使用的过程中，遇到一些小坑。特记录下来，方便以后自己查阅(未完待续…会有更多的坑在等着我的！)

1.在使用Toast的时候，出现“Uncaught ReferenceError: Toast is not defined”

  1. 引入

    import { Toast } from 'mint-ui';

  2. 代码

    Toast('提示信息');

按照官方文档，这样写就会出现“Uncaught ReferenceError: Toast is not defined”的错误

解决方案

  1. 引入

    import { Toast } from 'mint-ui';

  2. 代码

    this.$toast(this.msg);

  3. 效果

2.在使用MessageBox的时候，出现“Uncaught ReferenceError: MessageBox is not defined”

  1. 引入

    import { MessageBox } from 'mint-ui';

  1. 代码

    MessageBox.alert('提示', '操作成功');

解决方案

  1. 引入

    import { MessageBox } from 'mint-ui';

  2. 代码

    this.$messagebox.alert('提示', '操作成功');

  3. 效果
